# HJOW-Libs
HJOW's common libraries

# Requirements
Available on java 6 or above.
No third parties needed.

# Maven
```
<dependency>
    <groupId>org.duckdns.hjow</groupId>
    <artifactId>hjow-commons</artifactId>
    <version>0.0.25</version>
</dependency>
```
